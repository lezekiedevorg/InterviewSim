# Voix v2 — réponse push-to-talk : Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Permettre au candidat de répondre à la voix pendant l'entretien (push-to-talk), la reconnaissance ne faisant que remplir le champ de réponse existant.

**Architecture:** Un nouveau hook client `useSpeechRecognition` (miroir de `useSpeech`) wrappe l'API navigateur `SpeechRecognition`. Un helper pur `mergeTranscript` combine le texte tapé et le texte reconnu. `MeetingRoom` câble le tout ; `MeetingControls` gagne un bouton micro. Aucun changement de `page.tsx` (tout vit sous `MeetingRoom`).

**Tech Stack:** Next.js (App Router) + TypeScript, React hooks, Web Speech API (`SpeechRecognition`/`webkitSpeechRecognition`), vitest.

**Spec:** `docs/superpowers/specs/2026-07-03-voix-v2-design.md`

## Global Constraints

- **Copie UI en français** (l'app est francophone).
- **Aucune nouvelle dépendance** — API navigateur native uniquement.
- Composants navigateur = `"use client"` en tête de fichier.
- Tests via **vitest** : `npm test` (run unique). Descriptions de test en français, style de `tests/speech.test.ts`.
- Fallback obligatoire : `SpeechRecognition` absent → pas de bouton micro, la frappe clavier reste.
- Le bouton micro est désactivé pendant que le recruteur parle (`isSpeaking`).

```ts
// Combine le texte déjà tapé dans le champ avec le texte reconnu à la voix.
// ponytail: jointure par une seule espace ; suffisant pour du texte libre.
export function mergeTranscript(base: string, transcript: string): string {
  const t = transcript.trim();
  if (!t) return base;
  if (!base) return t;
  return base.replace(/\s+$/, "") + " " + t;
}
```

### Task 2: Hook `useSpeechRecognition`

**Files:**
- Create: `lib/useSpeechRecognition.ts`

**Interfaces:**
- Consumes: API navigateur `window.SpeechRecognition ?? window.webkitSpeechRecognition`.
- Produces: `useSpeechRecognition(): { supported: boolean; listening: boolean; transcript: string; start: () => void; stop: () => void; error: string | null }`.
  - `transcript` : texte agrégé (final + intermédiaire), remis à `""` à chaque `start()`.
  - `error` : message FR prêt à afficher, `null` si OK.

*Note : hook 100 % API navigateur → pas de test unitaire (comme `useSpeech` en v1). Vérification en Task 3 (navigateur).*

- [ ] **Step 1: Créer le hook**

Créer `lib/useSpeechRecognition.ts` :

```ts
"use client";

import { useCallback, useEffect, useRef, useState } from "react";

export function useSpeechRecognition() {
  const [supported, setSupported] = useState(false);
  const [listening, setListening] = useState(false);
  const [transcript, setTranscript] = useState("");
  const [error, setError] = useState<string | null>(null);
  // ponytail: types SpeechRecognition absents du lib DOM par défaut → any local.
  const recognitionRef = useRef<any>(null);

  useEffect(() => {
    if (typeof window === "undefined") return;
    const Ctor =
      (window as any).SpeechRecognition ?? (window as any).webkitSpeechRecognition;
    if (!Ctor) return;
    setSupported(true);

    const recognition = new Ctor();
    recognition.lang = "fr-FR";
    recognition.interimResults = true;
    recognition.continuous = true;

    recognition.onresult = (event: any) => {
      let text = "";
      for (let i = 0; i < event.results.length; i++) {
        text += event.results[i][0].transcript;
      }
      setTranscript(text);
    };
    recognition.onerror = (event: any) => {
      if (event.error === "not-allowed" || event.error === "service-not-allowed") {
        setError("Micro refusé — autorise l'accès au micro dans le navigateur.");
      } else if (event.error === "no-speech") {
        setError("Je n'ai rien entendu, réessaie.");
      } else {
        setError("Souci avec le micro, réessaie.");
      }
      setListening(false);
    };
    recognition.onend = () => setListening(false);

    recognitionRef.current = recognition;
    return () => {
      recognition.onresult = null;
      recognition.onerror = null;
      recognition.onend = null;
      recognition.stop();
    };
  }, []);

  const start = useCallback(() => {
    const r = recognitionRef.current;
    if (!r) return;
    setError(null);
    setTranscript("");
    try {
      r.start();
      setListening(true);
    } catch {
      // ponytail: start() throw si déjà démarré — sans gravité, on ignore.
    }
  }, []);

  const stop = useCallback(() => {
    recognitionRef.current?.stop();
    setListening(false);
  }, []);

  return { supported, listening, transcript, start, stop, error };
}
```
